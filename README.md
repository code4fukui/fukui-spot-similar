# fukui-spot-similar

A tool to find similar tourist spots in Fukui prefecture, Japan.

## Demo
https://code4fukui.github.io/fukui-spot-similar/

## Features
- Finds the top 3 most similar and dissimilar tourist spots based on text descriptions
- Displays detailed information and images for each spot
- Randomly selects a spot to display as the background

## Data / API
Uses the [Fukui Tourism Spot Open Data](https://github.com/code4fukui/fukui-spot) and the [txt2vec library](https://github.com/code4fukui/txt2vec) for text vectorization.
